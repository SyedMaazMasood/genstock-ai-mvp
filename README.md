# GenStock AI – MVP v1.0

Smart inventory + reordering for convenience stores (Wawa, 7-Eleven, bakeries).

**100% FREE** using:
- Llama 3.1 70B (via Groq)
- GPT-4o-mini (OpenAI)

### Features
- Upload shelf photo → auto-detect stock
- Smart Reordering Agent generates 3 drafts
- Human Approval Gate (one-tap approve)
- Export: WhatsApp text, JSON, PDF

### Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
